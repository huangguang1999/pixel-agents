# Third-Party Notices

## pablodelucca/pixel-agents (MIT)

This project bundles pixel-art assets (characters, floors, walls, furniture) and
rendering code derived from
[`pablodelucca/pixel-agents`](https://github.com/pablodelucca/pixel-agents)
under the terms of the MIT License reproduced below.

Specifically:
- `public/assets/` — character sprites, floor / wall / furniture PNG tiles and
  manifests.
- `src/office/` — Canvas 2D rendering engine, sprite cache, tile map,
  furniture catalog, character state machine, colorize (HSB tint), and
  related helpers.

Character sprites originate from JIK-A-4's *Metro City* top-down character
pack, used under the terms credited in the upstream repository.
